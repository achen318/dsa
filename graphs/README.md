# Graphs

| Algorithm                  | Time Complexity          |
| -------------------------- | ------------------------ |
| Breadth-First Search (BFS) | `O(\|V\|+\|E\|)`         |
| Depth-First Search (DFS)   | `O(\|V\|+\|E\|)`         |
| Dijkstra's                 | `O(\|V\|+\|E\|log\|V\|)` |

## Notes

- `|V|` - number of vertices/nodes
- `|E|` - number of edges/arcs
